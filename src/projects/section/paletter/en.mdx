### Key Features

- Chatbot-based text messaging service
- Automatic AI image generation matching the message content
- LLM-powered text generation, key sentence extraction, and image prompt suggestions
- Image editing functions (drawing, adding shapes, text removal, AI correction)
- Real-time Q&A through FAQ chatbot
- Message history/usage tracking, address book, and scheduled messaging

### Tech Stack (Frontend)

- **Framework**: Next.js
- **UI/UX**: Tailwind CSS, image editing libraries

### Tech Stack (Backend)

- **Framework**: Spring Framework
- **DB**: MariaDB
- **AI/LLM**: Flux 1.0, OpenAI LLM, **RAG (Retrieval-Augmented Generation)** based FAQ chatbot
- **Infrastructure**: Cloud deployment and server management
