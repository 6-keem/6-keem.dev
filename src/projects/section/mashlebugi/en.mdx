### Key Features

- **UI Improvement & Readability**

  - Simplified complex and unintuitive table format for easier information access
  - Provides timetable and remaining seat information in a clear and intuitive way

- **Quick Accessibility**

  - Highlights important data for faster access to necessary information

- **Convenient User Experience**
  - Intuitive design makes course registration and timetable checking easier

### Tech Stack

- **Framework**: React
- **Frontend**: TypeScript, Vite, Tailwind CSS
- **Data Storage**: Browser Local Storage

### Role

- **Project Planning & Development**
  - Designed and developed the Chrome extension
  - Parsed Hansung University Integrated Information System DOM and improved UI
  - Implemented browser extension features and local data storage
  - Applied intuitive design for enhanced usability
