### 주요 기능

- 챗봇 기반 문자 발송 서비스
- 문자 내용과 어울리는 AI 이미지 자동 생성
- LLM을 활용한 문자 생성, 주요 문장 추출, 이미지 프롬프트 추천
- 이미지 편집 기능 (그리기, 도형 추가, 텍스트 삭제, AI 보정)
- FAQ 챗봇을 통한 실시간 질의응답
- 문자 내역·사용량 조회, 주소록 및 예약 문자 관리

### 기술 스택 (프론트엔드)

- **프레임워크**: Next.js
- **UI/UX**: Tailwind CSS, 이미지 편집 라이브러리

### 기술 스택 (백엔드)

- **프레임워크**: Spring Framework
- **DB**: MariaDB
- **AI/LLM**: Flux 1.0, OpenAI LLM, **RAG (Retrieval-Augmented Generation)** 기반 FAQ 챗봇
- **인프라**: 서버 및 클라우드 환경 배포

### 역할

- **프로젝트 관리**
  - 프로젝트 일정 수립 및 관리
  - 팀원별 업무 분배 및 진행 상황 모니터링
- **프론트엔드 개발**
  - Next.js 기반 화면 구현 (FAQ, 문자 전송, 문자 내역 조회, 이미지 편집 등)
- **백엔드 개발**
  - Spring Framework 기반 API 설계 및 서버 로직 개발
  - 문자 발송, 이미지 생성/편집, FAQ 챗봇 기능 서버 연동
- **AI 모델 파인튜닝**
  - 이미지 생성·편집 모델 파인튜닝
  - 응답 정확도 개선 및 도메인 특화 지식 반영

### 프로젝트 기대 효과

- 문자와 이미지를 결합하여 메시지 전달력 강화
- LLM과 RAG 기반 지식 검색으로 정확한 정보 제공 및 사용자 만족도 향상
- FAQ 챗봇을 통한 24/7 실시간 응대 가능
- 이미지 생성·편집 자동화를 통한 사용자 맞춤형 메시지 제작 지원
- 프로젝트 관리 및 협업 경험을 통한 리더십과 문제 해결 능력 강화

### 성과

- **기업 사내 문제 해결형 캡스톤 프로젝트 우승**
- 실제 기업 요구사항 기반 서비스 기획·개발 → 실무 적용 가능성 검증
