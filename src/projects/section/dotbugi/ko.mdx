### 주요 기능

- **통합 강의/과제/퀴즈 정보 제공**  
  - 강의명, 일정, 교수 정보 등 강의 관련 데이터 한눈에 확인  
  - 과제 제목, 제출 기한, 요구 사항 등 과제 정보 통합 제공  
  - 퀴즈 제목, 진행 일정, 문제/답안 정보 조회 가능  
  - 출석 및 제출 상태 실시간 확인 가능  

- **제출 마감 알림 기능**  
  - 과제 및 퀴즈 제출 마감 임박 시 실시간 알림  
  - 중요한 일정 및 제출 기한을 놓치지 않도록 지원  

- **구글 캘린더 연동**  
  - 강의, 과제, 퀴즈 일정을 구글 캘린더와 동기화  
  - 인증 정보 안전 저장 및 일정 관리 자동화  

- **데이터 안전 관리**  
  - 수집된 데이터는 로컬 저장소에만 보관  
  - 외부 서버 전송 없이 개인정보 보호  
  - 최소한의 정보만 수집하며 보안 강화  

### 기술 스택

- **프레임워크**: React  
- **프론트엔드**: TypeScript, Vite, Tailwind CSS  
- **데이터 저장**: Browser Local Storage  
- **캘린더 연동**: Google Calendar API  
- **빌드 및 배포**: CI/CD 자동 릴리즈 및 자동 버저닝  

### 역할

- **프로젝트 기획 및 개발**  
  - 확장 프로그램 기획 및 아키텍처 설계  
  - 한성대학교 LMS 웹페이지 정보 수집 및 파싱  
  - 과제/퀴즈 마감 알림 기능 개발  
  - CI/CD 기반 자동 빌드, 릴리즈 및 버전 관리 구현
