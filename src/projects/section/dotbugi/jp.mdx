### 主な機能

- **講義・課題・クイズ情報の統合提供**

  - 講義名、スケジュール、教授情報の一括確認
  - 課題タイトル、提出期限、要件などを統合表示
  - クイズのタイトル、進行スケジュール、問題/回答情報を確認可能
  - 出席や提出状況のリアルタイム確認

- **提出期限通知機能**

  - 課題やクイズの締め切り直前にリアルタイム通知
  - 重要な予定や提出期限を見逃さないよう支援

- **Googleカレンダー連携**

  - 講義、課題、クイズのスケジュールをGoogleカレンダーと同期
  - 認証情報を安全に保存し、スケジュール管理を自動化

- **データ安全管理**
  - 収集データはローカルストレージのみで保存
  - 外部サーバーへの送信なしで個人情報を保護
  - 最小限の情報収集とセキュリティ強化

### 技術スタック

- **フレームワーク**: React
- **フロントエンド**: TypeScript, Vite, Tailwind CSS
- **データ保存**: ブラウザローカルストレージ
- **カレンダー連携**: Google Calendar API
- **ビルド・デプロイ**: CI/CDで自動リリースとバージョン管理

### 役割

- **プロジェクト企画・開発**
  - 拡張機能の企画とアーキテクチャ設計
  - 韓成大学LMSウェブページからの情報収集・パース
  - 課題・クイズ締め切り通知機能の開発
  - CI/CDを用いた自動ビルド・リリース・バージョン管理
