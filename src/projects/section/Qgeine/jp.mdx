### 主な機能

- **自然言語による SQL クエリ**

  - SQL を知らなくても自然言語でデータを検索可能
  - 非エンジニアでも簡単にデータ検索や分析が可能

- **ERD およびテーブルの可視化**

  - データベース構造から ERD を自動生成
  - テーブル属性（カラム名、型、制約など）を可視化

- **データ検索の利便性向上**

  - よく使うクエリを自動保存・再利用可能
  - クエリ結果を CSV/Excel に直接エクスポート

- **業務効率化**

  - 繰り返しのクエリ作成・実行を自動化
  - 分析や運用業務の時間を削減

- **安全なクエリ実行**
  - 危険なクエリを検証し安全なデータアクセスを実現
  - 実行前に結果を予測・確認しエラーを最小化

### 技術スタック

- **フレームワーク**: Electron, Vite
- **フロントエンド**: React, TypeScript, Tailwind CSS, Shadcn/UI
- **ビルド・デプロイ**: Electron Builder, GitHub Actions, GitHub Release
- **テスト**: Vitest, React Testing Library

### 役割

- **フロントエンド担当**
  - Electron + Vite によるデスクトップアプリ開発
- **ERD およびテーブル属性の可視化機能を開発**
  - DB 接続ウィザードおよび JDBC ドライバー自動化を実装

### プロジェクト期待効果

- SQL 知識がなくても迅速にデータ検索 → **データアクセス性の向上**
- ERD とカラム属性の可視化 → **データ構造理解の向上**
- 自動化による繰り返し作業削減 → **業務効率の向上**
- クエリ検証と安全装置により → **データエラーとリスクを最小化**
- Windows・macOS 両対応 → **ユーザー利便性の向上**
