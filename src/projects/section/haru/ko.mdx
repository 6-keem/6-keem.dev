### 주요 기능

- JLPT 레벨(N1~N5)별 단계적 단어 학습
- 카드 학습 (앞면: 한자 / 뒷면: 품사, 뜻, 예문, 원어민 TTS 발음)
- 4지선다 시험 기능 및 결과 리포트
- 북마크 단어 관리 및 재학습
- 대시보드 그래프(진도율, 정답률, 출석 현황)
- 학습 스트릭(연속 학습 기록) 시각화

### 기술 스택 (프론트엔드)

- **언어**: Java (Android)
- **프레임워크/도구**: Android Studio
- **라이브러리 및 API**: Retrofit2, Room, Google Mobile Service (GMS), Text-to-Speech, MPAndroidChart

### 기술 스택 (백엔드/ETL)

- **언어**: Python
- **라이브러리**: Selenium (단어 데이터 수집 자동화)

### 역할

#### **앱 전체 UI/UX 설계 및 구현**

- 카드 학습, 시험, 대시보드 화면 개발
- 색상 기반 출석/스트릭 시각화 적용

#### **TTS 기능 연동**

- Google TTS API를 활용해 원어민 발음 제공

#### **데이터 관리**

- Room DB를 활용한 단어/학습 결과 저장 및 로컬 관리

#### **데이터 수집 자동화**

- Python + Selenium으로 JLPT 단어/예문 데이터 크롤링 및 정제

### 프로젝트 기대 효과

- 단계별 학습을 통해 한자 단어에 점진적으로 익숙해짐
- 예문 기반 학습으로 단어 사용 맥락 파악 가능
- 원어민 발음을 통한 정확한 발음 학습
- 북마크와 통계를 통한 반복 학습 및 성취감 확보
