### Key Features

- Step-by-step JLPT vocabulary learning (N1~N5)
- Flashcards (Front: Kanji / Back: Part of speech, meaning, example sentence, native TTS pronunciation)
- Multiple-choice quiz function with result reports
- Manage bookmarked words and re-learn
- Dashboard graphs (progress rate, accuracy rate, attendance)
- Study streak visualization

### Tech Stack (Frontend)

- **Language**: Java (Android)
- **Framework/Tool**: Android Studio
- **Libraries & APIs**: Retrofit2, Room, Google Mobile Service (GMS), Text-to-Speech, MPAndroidChart

### Tech Stack (Backend/ETL)

- **Language**: Python
- **Library**: Selenium (automated word data collection)

### Role

#### **App-wide UI/UX Design & Implementation**

- Developed flashcards, quiz, and dashboard screens
- Applied color-based attendance/streak visualization

#### **TTS Integration**

- Provided native pronunciation using Google TTS API

#### **Data Management**

- Stored and managed vocabulary/learning results locally with Room DB

#### **Data Collection Automation**

- Collected and refined JLPT vocabulary/example sentence data with Python + Selenium

### Expected Impact

- Gradual exposure to kanji vocabulary through step-by-step learning
- Context understanding of vocabulary usage with example sentences
- Accurate pronunciation learning via native TTS
- Repetition learning and achievement tracking with bookmarks and statistics
