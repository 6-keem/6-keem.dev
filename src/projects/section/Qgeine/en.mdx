### Key Features

- **Natural Language SQL Query**

  - Query data using natural language without SQL knowledge
  - Enables non-developers to easily search and analyze data

- **ERD and Table Visualization**

  - Automatically generates ERDs from database structures
  - Visualizes table attributes (column names, types, constraints, etc.)

- **Enhanced Data Query Convenience**

  - Auto-save and reuse frequently used queries
  - Export query results directly to CSV/Excel

- **Work Efficiency**

  - Automates repetitive query writing and execution
  - Saves time in analysis and operations

- **Safe Query Execution**
  - Validates risky queries for safe access
  - Predicts and previews results to minimize errors
