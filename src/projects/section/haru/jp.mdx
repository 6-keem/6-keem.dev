### 主な機能

- JLPTレベル(N1~N5)別の段階的単語学習
- フラッシュカード（表: 漢字 / 裏: 品詞、意味、例文、ネイティブTTS発音）
- 四択テスト機能と結果レポート
- ブックマーク単語の管理と再学習
- ダッシュボードグラフ（進度率、正答率、出席状況）
- 学習ストリーク（連続学習記録）の可視化

### 技術スタック (フロントエンド)

- **言語**: Java (Android)
- **フレームワーク/ツール**: Android Studio
- **ライブラリ・API**: Retrofit2, Room, Google Mobile Service (GMS), Text-to-Speech, MPAndroidChart

### 技術スタック (バックエンド/ETL)

- **言語**: Python
- **ライブラリ**: Selenium（単語データ収集自動化）

### 役割

#### **アプリ全体のUI/UX設計と実装**

- フラッシュカード、テスト、ダッシュボード画面を開発
- 色ベースの出席・ストリーク可視化を適用

#### **TTS機能連携**

- Google TTS APIを活用してネイティブ発音を提供

#### **データ管理**

- Room DBを利用して単語/学習結果をローカル保存・管理

#### **データ収集自動化**

- Python + SeleniumでJLPT単語/例文データをクロール・整形

### プロジェクト期待効果

- 段階的学習で漢字単語に徐々に慣れる
- 例文ベースの学習で単語使用の文脈を把握
- ネイティブ発音で正しい発音を学習
- ブックマークと統計による繰り返し学習と達成感を確保
