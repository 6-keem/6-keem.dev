### Key Features

- **Integrated Lecture/Assignment/Quiz Info**

  - Quickly check lecture name, schedule, and professor information
  - View assignment title, due date, and requirements in one place
  - Access quiz title, schedule, and question/answer details
  - Real-time tracking of attendance and submission status

- **Deadline Notifications**

  - Receive real-time alerts for upcoming assignment and quiz deadlines
  - Never miss important dates or submission schedules

- **Google Calendar Integration**

  - Sync lecture, assignment, and quiz schedules with Google Calendar
  - Secure storage of authentication info and automated schedule management

- **Data Security Management**
  - All data stored locally only
  - Protects personal information with no external server transmission
  - Minimal data collection and enhanced security

### Tech Stack

- **Framework**: React
- **Frontend**: TypeScript, Vite, Tailwind CSS
- **Data Storage**: Browser Local Storage
- **Calendar Integration**: Google Calendar API
- **Build & Deployment**: CI/CD with automated release and versioning
