### 主な機能

- **UI 改善と可読性向上**

  - 従来の複雑で直感的ではない表形式を改善し、一目で情報を把握可能に
  - 時間割や履修残席数を分かりやすく提供

- **迅速なアクセス性**

  - 重要な情報を強調し、必要なデータを素早く確認可能

- **便利なユーザー体験**
  - 直感的なデザインにより、履修登録や時間割の確認がより容易に

### 技術スタック

- **フレームワーク**: React
- **フロントエンド**: TypeScript, Vite, Tailwind CSS
- **データ保存**: Browser Local Storage

### 役割

- **プロジェクト企画・開発**
  - 拡張機能の企画および UI/UX 設計
  - 韓城大学総合情報システムの DOM パースと UI 改善
  - ブラウザ拡張機能の実装およびローカルデータ保存
  - ユーザー利便性向上のための直感的デザインを適用
