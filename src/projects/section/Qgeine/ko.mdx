### 주요 기능

- **자연어 기반 SQL 질의**  
  - SQL을 몰라도 자연어 입력만으로 데이터 조회 가능  
  - 비개발자도 쉽게 데이터 검색 및 분석 수행  

- **ERD 및 테이블 시각화**  
  - 데이터베이스 테이블 구조를 ERD 형태로 자동 생성  
  - 테이블 속성(컬럼명, 타입, 제약조건 등) 시각화 제공  

- **데이터 조회 편의성 강화**  
  - 자주 쓰는 쿼리 자동 저장 및 재사용  
  - 조회 결과를 CSV/Excel로 바로 다운로드  

- **업무 효율화**  
  - 반복적 쿼리 작성·실행 과정을 자동화  
  - 분석 및 운영 업무 시간 절약  

- **안전한 쿼리 실행**  
  - 위험 쿼리 검증을 통한 안정적 데이터 접근  
  - 실행 전 결과 예측·확인으로 오류 최소화

### 기술 스택

- **프레임워크**: Electron, Vite  
- **프론트엔드**: React, TypeScript, Tailwind CSS, Shadcn/UI  
- **빌드 및 배포**: Electron Builder, GitHub Actions, GitHub Release  
- **테스트**: Vitest, React Testing Library

### 역할

- **프론트엔드 담당**  
  - Electron + Vite 기반 데스크탑 앱 개발  
  - **ERD 및 테이블 속성 시각화 기능 개발**  
  - DB 연결 마법사 및 JDBC 드라이버 자동화 구현

### 프로젝트 기대 효과

- SQL 지식이 없어도 데이터를 빠르게 조회 → **데이터 접근성 향상**  
- ERD와 컬럼 속성 시각화를 통해 → **데이터 구조 이해도 향상**  
- 반복 작업을 줄이고 자동화를 통해 → **업무 효율성 증가**  
- 쿼리 검증 및 안전 장치로 → **데이터 오류 및 리스크 최소화**  
- 하나의 앱으로 Windows, macOS 지원 → **사용자 편의성 강화**

### 성과 (Achievement)

- **기획부터 개발·배포까지 단독 수행**하며 완성도 높은 데스크탑 앱 구현  
- JDBC 드라이버 자동 설치 및 검증 기능으로 **사용자 설치 편의성 향상**  
- ERD 및 테이블 속성 시각화를 통해 **데이터 구조 가시성 제공**  
- GitHub Actions 기반 자동 배포 파이프라인 구축으로 **배포 효율 극대화**
