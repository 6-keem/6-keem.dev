### 주요 기능

- 보호자-피보호자 연동 및 원터치 연락 기능
- 실시간 위치 공유 및 보호자에 의한 경로 설정
- 사고 다발 지역 데이터 기반 경로 안내 (우회 기능 포함)
- 캘린더를 통한 이동 일정 예약 및 관리

### 기술 스택 (프론트엔드)

- **프레임워크**: Jetpack Compose (Android Native)

### 기술 스택 (백엔드)

- **프레임워크**: Spring Framework
- **DB**: MariaDB

### 역할

- 일정 관리 및 캘린더 기능 구현
- 사고 다발 지역 데이터를 기반으로 경로 표시 기능 개발

### 프로젝트 기대 효과

- 사회적 약자의 이동 안전성 향상: 사고 다발 지역을 자동으로 회피하는 경로 안내로 위험 노출을 최소화
- 보호자 안심 서비스 제공: 보호자가 실시간으로 피보호자의 위치를 확인하고, 긴급 시 즉시 연락 가능
- 생활 편의성 증대: 이동 일정 예약 및 알림 기능으로 반복 이동의 효율성 개선
- 지역 사회 연계 가능성: 공공 데이터(사고 다발 구역 등)를 활용해 지자체 안전 서비스와의 연동 확장성 확보
