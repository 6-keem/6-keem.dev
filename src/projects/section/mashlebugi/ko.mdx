### 주요 기능

- **UI 개선 및 가독성 향상**  
  - 기존 복잡하고 직관적이지 않은 표 형식을 개선하여 한눈에 정보를 파악 가능  
  - 시간표 및 수강 잔여 인원 정보를 직관적이고 보기 쉽게 제공  

- **빠른 접근성**  
  - 주요 정보를 강조하여 사용자가 필요한 데이터 빠르게 확인 가능  

- **편리한 사용 경험**  
  - 직관적인 디자인 적용으로 수강 신청 및 시간표 조회가 더욱 용이

### 기술 스택

- **프레임워크**: React  
- **프론트엔드**: TypeScript, Vite, Tailwind CSS  
- **데이터 저장**: Browser Local Storage

### 역할

- **프로젝트 기획 및 개발**  
  - 확장 프로그램 기획 및 UI/UX 설계  
  - 한성대학교 종합정보시스템 페이지 DOM 파싱 및 UI 개선  
  - 브라우저 확장 기능 구현 및 데이터 로컬 저장  
  - 사용자 편의성을 위한 직관적 디자인 적용
